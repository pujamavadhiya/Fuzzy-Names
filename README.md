# Fuzzy Name conversion of Hindi Names in Police Records

### Installation Steps

1. Clone the repository.
   ```bash
    git clone https://github.com/pujamavadhiya/Fuzzy-Names.git
2. Navigate to the project directory.
   ```bash
    cd Fuzzy-Names
3. Install the dependencies:
   ```bash
   npm install
4. Run:
   ```bash
   npm run dev
